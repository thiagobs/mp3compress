# mp3compress
A tiny mp3 file compressor using ffmpeg binary

## Installation

```
$ git clone https://github.com/thiagobs/mp3compress.git
$ cd mp3compress
$ npm install -g .
```

## Usage

```
$ mp3compress --input /dir/of/input/mp3/files --output /dir/of/output/mp3/files [--bitrate=128k] [--force]
```
